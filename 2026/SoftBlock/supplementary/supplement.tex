\section{Pipeline Diagram}
\label{sec:diagram}

The SoftBlock pipeline consists of the following stages:
\begin{enumerate}
    \item \textbf{Reference PPI} ($G_{\text{ref}}$, STRING) $\to$ SoftBlock soft memberships $R \in [0,1]^{|V_{\text{ref}}| \times K}$
    \item \textbf{Target PPI} ($G$: BioPlex / HuRI / ...) $\to$ Overlapping blocks $\{B_k\}_{k=1}^K$
    \item Local solver in each block (MCL / wrappers)
    \item Candidate pool (union + dedupe)
    \item Graph-only scoring (+ optional learned rerank)
    \item Top-$N$ selection + overlap suppression $\to$ Predicted complexes
\end{enumerate}
